# 【算法】模拟类

## 1 二维数据坐标模拟

此类题目基本上是考察坐标映射公式，主要分为四种对称：

- 上下对称：`(i, j) -> (n - i - 1, j)`
- 左右对称：`(i, j) -> (i, n - j - 1)`
- 主对角线对称：`(i, j) -> (j, i)`
- 副对角线对称：`(i, j) -> (n - j - 1, n - i - 1)`
